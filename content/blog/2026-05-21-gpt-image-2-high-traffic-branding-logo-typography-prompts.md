# GPT Image 2高赞品牌字标与Logo海报提示词

如果只看传播面和搜索意图，品牌字标、Logo 海报、字体实验和概念广告也很容易起量。

它们的好处是适合做封面、首页 Hero、品牌落地页和社交图，既能看审美，也能直接抄结构。

## 1. 品牌字体

- 用法：字标海报、品牌视觉、logo 概念图
- 标签：`branding / illustration / poster / typography`
- 样例图：![品牌字体](https://github.com/denghuichao/gpt4o-image-prompts/blob/master/images/316.jpeg?raw=true)
- Prompt 摘要：

```text
创建一个形状像 {OBJECT} 的印刷插图，其中文本本身形成形状，大胆而俏皮的字体风格填充整个轮廓。
```

- 源链接：<https://x.com/aziz4ai/status/1940348752969322544>

## 2. 玻璃变形海报

- 用法：品牌海报、字标实验、折射视觉
- 标签：`branding / poster / typography`
- 样例图：![玻璃变形海报](https://github.com/denghuichao/gpt4o-image-prompts/blob/master/images/147.jpeg?raw=true)
- Prompt 摘要：

```text
设计一张带有“[BRAND NAME]”字样的电影式品牌海报，在下面包括一个粗体的六字标语，应用 3D 玻璃折射效果。
```

- 源链接：<https://x.com/aziz4ai/status/1936950051856941470>

## 3. 吃掉你的文字

- 用法：logo 海报、食品品牌、字形概念图
- 标签：`branding / food / logo / minimalist / product / typography`
- 样例图：![吃掉你的文字](https://github.com/denghuichao/gpt4o-image-prompts/blob/master/images/209.jpeg?raw=true)
- Prompt 摘要：

```text
创建由其真实质感完全构成，并使用与其最具有标志性的品牌相联系的标识进行风格的“[单词]”。
```

- 源链接：<https://x.com/aziz4ai/status/1935749368876867967>

## 4. 品牌设计指南海报

- 用法：品牌首发、视觉规范页、产品页头图
- 标签：`branding / minimalist / poster / product / typography`
- 样例图：![品牌设计指南海报](https://github.com/denghuichao/gpt4o-image-prompts/blob/master/images/62.jpeg?raw=true)
- Prompt 摘要：

```text
Create a vertical 9:16 brand design guide poster using the uploaded product image. Adapt the design style to match the product’s niche and visual identity.
```

- 源链接：<https://x.com/ai4everyday/status/1925838516979646795>

## 5. 超现实主义极简概念广告

- 用法：概念广告、品牌 KV、极简产品海报
- 标签：`branding / landscape / minimalist / nature / poster / product`
- 样例图：![超现实主义极简概念广告](https://github.com/denghuichao/gpt4o-image-prompts/blob/master/images/273.jpeg?raw=true)
- Prompt 摘要：

```text
“{品牌或产品名称}”——超现实主义极简概念广告。创建 1:1 高分辨率海报，使用最少的元素和象征性的故事讲述。
```

- 源链接：<https://x.com/aziz4ai/status/1927459345790288258>
